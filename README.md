# ICT Home PROS Payment Program

Version 1.0

This is the starting web app shell for the ICT Home PROS Payment Program. The first screen is a branded login page using the ICT Home Pros logo.

## Current Build

- Static web app starter
- Login screen with username and password fields
- Local project assets stored in `assets/`

## Security Note

The current login page is only the front-end screen. Real username and password protection should be connected to a backend before this app is used for live payment workflows.
